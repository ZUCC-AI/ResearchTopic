# TAAD(Textual Adversarial Attack and Defense)

18级**楼学长**和16级**张学长**曾经一起参与过的课题。

文本攻击与防御，用于检测当前深度学习模型安全性的一个topic,分为：

- 攻击：通过黑盒、白盒的攻击方法，揭露深度学习模型的漏洞
- 防御：对攻击中发现的漏洞进行模型鲁棒性能针对性提升
